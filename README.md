# Caava Widget Next.js Test

A Next.js app to test the Caava AI Widget.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (usually `http://localhost:3000`)

## Build

To build for production:
```bash
npm run build
```

To start the production server:
```bash
npm start
```

The built files will be in the `.next` folder.
